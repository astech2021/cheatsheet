HTML and CSS References

HTML TAGS

HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content

Unclosed Tags in Html
Tag Tag Name Tag Description
< br > Break It breaks the line of the code.
< hr > Horizontal line This tag is used to put a line across the webpage.
HTML Meta Tags
< Doctype > Doctype This tag tells the browser about what document type to expect.
< title > Style This tag defines the title of the document.
< link > Link This tag defines the relationship between the current document and an external resource.
< meta > Metadata This tag defines metadata about an HTML document. Metadata is data (information) about data.
< style > Style This tag is used to define style information (CSS) for a document.
Below are some HTML Text Tags
< p > Paragraph This tag defines a paragraph.
< strong > Strong This tag is used to define text with strong importance. The content inside is typically displayed in bold.
< em > Emphasizes This tag is used to define emphasized text. The content inside is typically displayed in italic.
< abbr > Abbreviation This tag defines an abbreviation or an acronym, like "HTML", "CSS".
< address > Address This tag defines the contact information for the author/owner of a document or an article.
HTML Image and Object Tags
< img > Image This tag is used to embed an image in an HTML page.
< area > Area This defines an area inside an image map (an image map is an image with clickable areas).
< map > Map This tag is used to define an image map.
< param > Parameter This tag is used to define parameters for an object element.
< object > Object This tag defines a container for an external resource.
HTML list tags
< ul > Unordered List This tag defines an unordered (bulleted) list.
< ol > Ordered List This tag defines an ordered (numbers) list.
< ol > Ordered List This tag defines an ordered (numbers) list.
< dl > Description List This tag defines a description list.
< dt > Description List This tag defines a term/name in a description list.

CSS PROPERTIES

The following section contains some CSS properties belonging to the latest CSS3 specifications

CSS Properties
Property Description
align-content Specifies the alignment of flexible container's items within the flex container.
align-items Specifies the default alignment for items within the flex container.
align-self Specifies the alignment for selected items within the flex container.
animation Specifies the keyframe-based animations.
animation-delay Specifies when the animation will start
animation-direction Specifies whether the animation should play in reverse on alternate cycles or not.
animation-duration Specifies the number of seconds or milliseconds an animation should take to complete one cycle.
background Defines a variety of background properties within one declaration.
background-attachment Specify whether the background image is fixed in the viewport or scrolls.
background-image Defines an element's background image.
background-color Defines an element's background color.
border Sets the width, style, and color for all four sides of an element's border.
border-bottom Sets the width, style, and color of the bottom border of an element.
border-color Sets the color of the border on all the four sides of an element.
font-weight Specify the font weight of the text.
justify-content Specifies how flex items are aligned along the main axis of the flex container after any flexible lengths and auto margins have been resolved.
font-family Defines a list of fonts for element.
float Defines a variety of font properties within one declaration.
margin Sets the margin on all four sides of the element.
